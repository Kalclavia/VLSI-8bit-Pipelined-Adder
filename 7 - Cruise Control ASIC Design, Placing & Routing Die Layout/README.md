# Lab 7: Cruise-Control ASIC Place and Route

This lab introduces ASIC implementation flow concepts using a cruise-control digital design. It focuses on netlist-level implementation, constraints, timing reports, place-and-route output, and physical-design metrics.

## Contents

- `Lab 7 Report.pdf` and `Lab 7 Report.docx`: Formal lab report documenting the ASIC implementation exercise.
- `PartA/`: Verilog netlist, area report, constraint report, and supporting screenshot.
- `PartB/`: Timing reports including max paths, min paths, hold-time violator report, and violation summary.
- `PartC/`: Die layout capture, standard-cell count image, and wirelength/via summary image.

## What This Demonstrates

- ASIC implementation workflow beyond schematic and custom-cell design.
- Review of synthesized or netlist-level digital logic.
- Use of area and constraint reports to evaluate implementation quality.
- Timing-analysis concepts including max paths, min paths, and hold-time violations.
- Physical-design review of placed/routed layout, standard-cell usage, routing, wirelength, and vias.

## Review Focus

Start with `Lab 7 Report.pdf`, then review the `PartA/`, `PartB/`, and `PartC/` artifacts. This lab is useful for demonstrating familiarity with implementation outputs that are common in digital ASIC and physical-design workflows.
