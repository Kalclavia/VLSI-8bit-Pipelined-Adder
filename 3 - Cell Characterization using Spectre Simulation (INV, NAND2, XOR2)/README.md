# Lab 3: Cell Characterization Using Spectre Simulation

This lab characterizes the primitive CMOS cells used in the project. The inverter, NAND2, and XOR2 cells are simulated in Cadence Spectre to evaluate their behavior and establish timing and performance data for later hierarchical designs.

## Contents

- `Lab 3 Report.pdf` and `Lab 3 Report.docx`: Formal lab report documenting simulation setup, characterization results, and analysis.
- Spectre-based characterization work for inverter, NAND2, and XOR2 cells.

## What This Demonstrates

- Cadence Spectre simulation workflow.
- Functional verification of transistor-level logic cells.
- Measurement and interpretation of delay behavior.
- Cell-level characterization as a prerequisite for larger digital systems.
- Analytical connection between layout, transistor sizing, load behavior, and circuit performance.

## Review Focus

Start with `Lab 3 Report.pdf` to understand how the primitive cells were simulated and characterized. This lab provides the measurement discipline that supports later delay, power, and optimization analysis.
