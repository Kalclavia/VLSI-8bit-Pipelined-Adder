# Lab 2: Gate Silicon Layouts

This lab moves from schematic-level logic into custom CMOS physical layout. The focus is on implementing primitive digital logic gates as silicon layouts, including inverter, NAND2, and XOR2 cells.

## Contents

- `Lab 2 Report.pdf` and `Lab 2 Report.docx`: Formal lab report documenting the layout work and results.
- Custom layout work for inverter, NAND2, and XOR2 primitive cells.

## What This Demonstrates

- Transistor-level CMOS layout of basic digital logic gates.
- Understanding of pull-up and pull-down network implementation.
- Physical-design awareness for diffusion, polysilicon, metal routing, contacts, and cell structure.
- Translation of Boolean logic into manufacturable layout geometry.
- Foundation for building larger arithmetic blocks from characterized primitive cells.

## Review Focus

Review `Lab 2 Report.pdf` for the cell-layout methodology and design decisions. This lab is important because it connects digital logic behavior to the physical structures used later in the adder layouts.
