# Lab 6: D Flip-Flop Silicon Layout and Simulation

This lab develops a D flip-flop, adding sequential logic to the project. The flip-flop supports the pipelined adder design by providing state-holding elements for datapath staging.

## Contents

- `Lab 6 Report.pdf` and `Lab 6 Report.docx`: Formal lab report documenting the D flip-flop layout, simulation, and results.

## What This Demonstrates

- Sequential CMOS circuit design in Cadence.
- Layout and simulation of a clocked storage element.
- Understanding of state, timing, and datapath pipelining requirements.
- Integration of sequential logic concepts with arithmetic datapath design.
- Verification of a timing-dependent digital circuit.

## Review Focus

Review `Lab 6 Report.pdf` for the D flip-flop design and simulation results. This lab is important for understanding how the overall adder design moves beyond pure combinational logic into pipelined digital architecture.
