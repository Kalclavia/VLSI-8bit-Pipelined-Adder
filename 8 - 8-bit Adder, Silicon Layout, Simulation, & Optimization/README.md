# Lab 8: 8-bit Adder Silicon Layout, Simulation, and Optimization

This lab is the main culmination of the adder design sequence. It brings together the earlier schematic, layout, simulation, sequential logic, and optimization work into a larger 8-bit adder implementation.

## Contents

- `Lab 8 Report.pdf` and `Lab 8 Report.docx`: Formal lab report documenting the 8-bit adder layout, simulation, and optimization work.

## What This Demonstrates

- Hierarchical scaling from 1-bit and 4-bit arithmetic blocks to an 8-bit design.
- Larger digital-layout organization in Cadence.
- Simulation-based verification of a multi-bit arithmetic circuit.
- Optimization workflow for performance, power, or implementation quality.
- Ability to connect cell-level design decisions to a larger datapath implementation.
