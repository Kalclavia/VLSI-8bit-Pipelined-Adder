# Lab 5: 4-bit Adder Silicon Layout and Simulation

This lab scales the arithmetic design from a 1-bit adder to a 4-bit adder. It emphasizes hierarchical composition, layout reuse, and simulation of a wider combinational arithmetic block.

## Contents

- `Lab 5 Report.pdf` and `Lab 5 Report.docx`: Formal lab report documenting the 4-bit adder layout, simulation setup, and results.

## What This Demonstrates

- Hierarchical digital design using smaller verified building blocks.
- Multi-bit arithmetic implementation in Cadence.
- Layout and simulation workflow for a larger combinational block.
- Practical scaling from cell-level logic to datapath-level design.
- Verification discipline across increasing design complexity.

## Review Focus

Review `Lab 5 Report.pdf` for the 4-bit adder implementation and simulation results. This lab is a bridge between the detailed 1-bit adder work and the final 8-bit pipelined adder.
